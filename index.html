🔇 Suara belum aktif
Firefox memerlukan interaksi pertama untuk mengaktifkan mesin suara. Klik tombol di samping untuk mengaktifkan.
❌ Suara tidak ditemukan. Kemungkinan penyebab:
  • Firefox di Linux: install paket TTS dengan perintah:
    sudo apt install speech-dispatcher lalu restart Firefox.
  • Firefox di Windows: pastikan Windows Speech/Narrator sudah diaktifkan di Settings → Time & Language → Speech.
  • Alternatif: gunakan Chrome atau Edge.
